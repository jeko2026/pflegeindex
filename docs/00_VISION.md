# Directory Platform

## Видение проекта

### Миссия

Создать универсальную платформу для построения профессиональных каталогов, ориентированных на пользователей, поисковые системы и долгосрочное развитие.

Мы создаём не отдельный сайт, а систему, которая позволит быстро запускать новые каталоги на единой архитектуре.

---

# Главная идея

PflegeIndex — это первый проект платформы.

Он служит проверкой архитектурных решений, которые в дальнейшем будут использоваться в других каталогах.

Каждое принятое решение должно рассматриваться не только с точки зрения PflegeIndex, но и с точки зрения всей платформы.

---

# Наша цель

Создать платформу, на которой можно запускать новые каталоги без переписывания ядра системы.

Примеры будущих проектов:

- PflegeIndex
- BestatterIndex
- FriedhofIndex
- ÄrzteIndex
- HandwerkerIndex
- ImmobilienIndex
- HotelIndex

и другие.

# Что считается успехом

Успехом считается не количество написанного кода.

Успехом считается возможность использовать существующие модули повторно при создании следующего каталога.

Каждый новый проект должен становиться проще в разработке благодаря уже существующей платформе.

---

# Основной принцип

Любая новая возможность должна усиливать платформу, а не только текущий проект.

Если функциональность может быть реализована как часть платформы — она должна реализовываться именно там.

Если функциональность относится исключительно к конкретному каталогу — она должна оставаться в проектном модуле.

---

# Долгосрочная цель

Создать зрелую Directory Platform, которая станет основой для множества независимых каталогов, использующих единое ядро, общую архитектуру и единые принципы разработки.

PflegeIndex — первый шаг на этом пути.
